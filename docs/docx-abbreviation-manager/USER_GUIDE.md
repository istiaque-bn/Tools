# User guide

1. Open **DOCX Abbreviation Manager** from the dashboard.
2. Choose Abbreviate or Deabbreviate.
3. Select a profile, replacement policy, content areas, and optional glossary behavior.
4. Upload one `.docx` file. Other formats, macros, encryption, external relationships, unsafe embedded files, and suspicious ZIP packages are rejected.
5. Select **Analyse document**.
6. Review highlighted suggestions. Filter by text, status, category, or ambiguity.
7. Accept, reject, edit, reset, undo, or redo decisions. Select a meaning before accepting an ambiguous expansion.
8. Resolve or reject every ambiguous pending suggestion.
9. Generate the DOCX. Only accepted and edited suggestions are applied.
10. Download the processed DOCX once. Temporary files are then deleted.

Cancel at any time to delete the session. Sessions also expire automatically.
